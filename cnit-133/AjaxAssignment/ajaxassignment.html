Explanation of AJAX

When the 'Get CD collection' button is clicked, the 'loadDoc' function is called. Within this function, an XMLHttpRequest object is created to communicate with the server. The 'onload' of the XMLHttpRequest object is set to a callback function named 'myFunction', which will excute when the response is ready. The main task of 'myFunction' is to parse the XML data and display it in an HTML table.The open() method creates a GET request with a url to the XML file. And the send() method sends the request to the server.

GoodsForYou E-commerce Website

GoodsForYou, a new E-commerce company, has spent over 90 percent of its budget to develop its website. Given the limited remaining funds, hiring a team to build a secure payment system is financially impractical. To address this, I propose leveraging the Unified Checkout Solutions from Mastercard. This API not only supports Mastercard but also accommodates Visa, American Express, and Discover Cards. Futhermore, the API is free to use, presenting a cost-effective solution for our payment system.

The Unified Checkout Solutions API utilizes OAuth 1.0 to generate a 2048-bit private RSA key for both the Signing Key, used to authenticate API calls, and the Cosumer Key, which identifies the client application. To ensure the security of sensitive payment data during transport, the API employs JSON Web Encryption (JWE), which encrypt a JSON file.

Using the Unified Checkout Solutions API will not only presents a cost-eddective solution but also save us resources, which can be allocate to something else.
