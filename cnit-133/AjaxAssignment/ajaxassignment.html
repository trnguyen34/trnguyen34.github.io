Explanation of AJAX

When the 'Get CD collection' button is clicked, the 'loadDoc' function is called. Within this function, an XMLHttpRequest object is created to communicate with the server. The 'onload' of the XMLHttpRequest object is set to a callback function named 'myFunction', which will excute when the response is ready. The main task of 'myFunction' is to parse the XML data and display it in an HTML table.The open() method creates a GET request with a url to the XML file. And the send() method sends the request to the server.
