Explanation of AJAX

When the 'Get CD collection' button is clicked, the 'loadDoc' function is called. Within this function, an XMLHttpRequest object is created to communicate with the server. The 'onload' of the XMLHttpRequest object is set to a callback function named 'myFunction', which will excute when the response is ready. The main task of 'myFunction' is to parse the XML data and display it in an HTML table.The open() method creates a GET request with a url to the XML file. And the send() method sends the request to the server.

GoodsForYou E-commerce Website

GoodsForYou is a new E-commerce company that just spent more than 90 percent of their budget on developing the website. Due to the lack of fund left, we cannot not afford to hire a team of developers to build a secure pay system. Therefore, I recommend we should use the Unified Checkout Solutions API from Mastercard. This API does not just suport Mastercard but it also supports Visa, American Express, and Discover cards. Additionally, the API is free to use.

The Unified Checkout Solutions API uses the OAuth1.0. The OAuth1.0 needs to be included in the header in order to authenticate API calls and identifies the client application. For senstive payment data, it will be using JSON Web Encryption (JWE).
